# Mern Stack

## Description
Creating a website using MongoDb, Express, React, Nodejs<br />
User is able to create a record for employee<br />

## Updateable Records
Name<br />
Position<br />
Salary<br />
Role<br />

## Features
Able to edit the records<br />
Able to delete the roles<br />
Stored in MongoDb for future use<br />
